# AAPL Quantitative Pricing Tool

## Overview

This project presents a Python-based quantitative finance workflow using Apple Inc. (AAPL) as a case study.

The objective is to analyse historical stock prices, simulate future price movements, and price European options using quantitative finance models.

---

## Methods

- Historical stock price analysis
- Log return calculation
- Augmented Dickey-Fuller (ADF) stationarity test
- ARIMA(1,0,0) model
- Geometric Brownian Motion (GBM)
- Monte Carlo Simulation (10,000 paths)
- European Call Option Pricing
- European Put Option Pricing
- Black-Scholes Validation
